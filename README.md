dsd
===

A small statusbar updater in ruby

## Configuration
Configuration is read from a yaml file at ~/.dsd.conf
See dot.dsd.conf.example for an example.

## Requirements
 * eventmachine
 * ffi
 * libxname (https://github.com/Igneous/libxname)
 * trollop
 * daemonize
